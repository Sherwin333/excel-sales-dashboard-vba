# 📊 Excel Sales Dashboard with VBA Automation

## 🚀 Overview

This project is an automated Excel-based sales reporting dashboard built using VBA macros. It processes raw sales data and generates monthly summaries, KPIs, and insights with a single click.

---

## 🔧 Features

* Automated monthly sales report generation
* KPI calculations (Revenue, Target, Variance, % Achievement)
* Region-wise and category-wise analysis
* Best month and top category detection
* Clean formatted output with popup summary

---

## 📁 Files Included

* `SalesReport_WithMacro.xlsm` → Main Excel dashboard with macro support
* `ReportMacro.bas` → VBA macro code

---

## ▶️ How to Use

1. Open the `.xlsm` file in Excel
2. Enable macros
3. Go to **Developer → Macros → Run GenerateMonthlyReport**
4. View generated report in "Monthly Summary" sheet
